# Nora Portfolio Starter

This is a multi-page Apple-inspired portfolio starter with:
- cinematic scrolling feel via Lenis
- GSAP + ScrollTrigger animations
- multi-page structure
- premium glassmorphism cards
- project case-study layout

## Files
- index.html
- about.html
- projects.html
- project-helmholtz.html
- research.html
- resume.html
- contact.html
- styles.css
- script.js

## How to use
1. Open `index.html` locally to preview.
2. Replace placeholder text with your real content.
3. Replace placeholder media blocks with your own images.
4. Upload the whole folder to GitHub Pages, Netlify, or Vercel.

## Easy customizations
- Change colors in `:root` inside `styles.css`
- Update name and nav text in each HTML file
- Replace placeholder sections with your real projects
- Add your real links on `contact.html`
